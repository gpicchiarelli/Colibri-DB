Uso Base

coldb [OPTIONS] [COMMAND]

Opzioni Globali

Opzione Descrizione Default
--config, -c File di configurazione colibridb.conf.json
--verbose, -v Output verboso false
--help, -h Mostra help false
--version Mostra versione false

Comandi Database

start

Avvia il database

coldb start [OPTIONS]

Opzioni

  • --daemon, -d - Avvia come daemon
  • --pid-file - File PID per daemon
  • --log-file - File di log

Esempi

# Avvia database
coldb start

# Avvia come daemon
coldb start --daemon --pid-file /var/run/colibridb.pid

# Avvia con log personalizzato
coldb start --log-file /var/log/colibridb.log

stop

Ferma il database

coldb stop [OPTIONS]

Opzioni

  • --force, -f - Forza shutdown
  • --timeout - Timeout in secondi

Esempi

# Ferma database
coldb stop

# Forza shutdown
coldb stop --force

# Shutdown con timeout
coldb stop --timeout 30

status

Mostra stato del database

coldb status [OPTIONS]

Opzioni

  • --json - Output in formato JSON
  • --metrics - Mostra metriche dettagliate

Esempi

# Stato base
coldb status

# Stato con metriche
coldb status --metrics

# Output JSON
coldb status --json

Comandi Schema

create-table

Crea una tabella

coldb create-table TABLE_NAME [OPTIONS]

Opzioni

  • --columns - Definizione colonne (JSON)
  • --primary-key - Chiave primaria
  • --file - File con definizione schema

Esempi

# Crea tabella semplice
coldb create-table users --columns '{"id":"int","name":"string"}' --primary-key id

# Crea tabella da file
coldb create-table users --file schema.json

drop-table

Elimina una tabella

coldb drop-table TABLE_NAME [OPTIONS]

Opzioni

  • --force, -f - Forza eliminazione
  • --cascade - Elimina dipendenze

Esempi

# Elimina tabella
coldb drop-table users

# Elimina con dipendenze
coldb drop-table users --cascade

list-tables

Lista le tabelle

coldb list-tables [OPTIONS]

Opzioni

  • --json - Output in formato JSON
  • --schema - Mostra schema dettagliato

Esempi

# Lista tabelle
coldb list-tables

# Con schema dettagliato
coldb list-tables --schema

# Output JSON
coldb list-tables --json

Comandi Dati

insert

Inserisci dati

coldb insert TABLE_NAME [OPTIONS]

Opzioni

  • --data - Dati in formato JSON
  • --file - File con dati
  • --batch - Inserimento batch

Esempi

# Inserisci singolo record
coldb insert users --data '{"id":1,"name":"Alice"}'

# Inserisci da file
coldb insert users --file data.json

# Inserimento batch
coldb insert users --file data.json --batch

query

Esegui query

coldb query "SQL_QUERY" [OPTIONS]

Opzioni

  • --output, -o - File di output
  • --format - Formato output (table, json, csv)
  • --limit - Limite risultati

Esempi

# Query semplice
coldb query "SELECT * FROM users"

# Query con formato
coldb query "SELECT * FROM users" --format json

# Query con limite
coldb query "SELECT * FROM users" --limit 100

Comandi Amministrazione

backup

Crea backup

coldb backup [OPTIONS]

Opzioni

  • --output, -o - Directory di output
  • --compress - Comprimi backup
  • --incremental - Backup incrementale

Esempi

# Backup completo
coldb backup --output /backup/colibridb

# Backup compresso
coldb backup --output /backup/colibridb --compress

# Backup incrementale
coldb backup --output /backup/colibridb --incremental

restore

Ripristina backup

coldb restore BACKUP_PATH [OPTIONS]

Opzioni

  • --force, -f - Forza ripristino
  • --tables - Ripristina solo tabelle specifiche

Esempi

# Ripristina backup
coldb restore /backup/colibridb

# Ripristina tabelle specifiche
coldb restore /backup/colibridb --tables users,orders

vacuum

Esegui VACUUM

coldb vacuum [OPTIONS]

Opzioni

  • --table - Tabella specifica
  • --analyze - Aggiorna statistiche
  • --full - VACUUM completo

Esempi

# VACUUM completo
coldb vacuum

# VACUUM tabella specifica
coldb vacuum --table users

# VACUUM con analisi
coldb vacuum --analyze

Comandi Monitoring

monitor

Monitora database in tempo reale

coldb monitor [OPTIONS]

Opzioni

  • --interval - Intervallo aggiornamento (secondi)
  • --metrics - Metriche da mostrare
  • --export - Esporta metriche

Esempi

# Monitora base
coldb monitor

# Monitora con intervallo personalizzato
coldb monitor --interval 5

# Monitora metriche specifiche
coldb monitor --metrics throughput,latency,memory

stats

Mostra statistiche

coldb stats [OPTIONS]

Opzioni

  • --json - Output in formato JSON
  • --reset - Reset statistiche
  • --export - Esporta statistiche

Esempi

# Statistiche base
coldb stats

# Output JSON
coldb stats --json

# Reset statistiche
coldb stats --reset

Comandi Configurazione

config

Gestisci configurazione

coldb config [COMMAND] [OPTIONS]

Sottocomandi

  • show - Mostra configurazione
  • set - Imposta parametro
  • get - Ottieni parametro
  • validate - Valida configurazione

Esempi

# Mostra configurazione
coldb config show

# Imposta parametro
coldb config set bufferPoolSize 2000

# Ottieni parametro
coldb config get bufferPoolSize

# Valida configurazione
coldb config validate
