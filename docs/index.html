🐦 Colibrì DB

RDBMS sperimentale ad alte prestazioni per macOS

Un database relazionale moderno scritto in Swift 6.2, progettato per gestire milioni di connessioni logiche con un'architettura modulare: motore heap su disco con WAL, MVCC, indici pluggabili e CLI amministrativa.

Swift 6.2 macOS 13+ Apple Silicon WAL • MVCC • B+Tree
