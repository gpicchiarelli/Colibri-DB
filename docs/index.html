🐦 Colibrì DB

RDBMS sperimentale ad alte prestazioni per macOS

Un database relazionale moderno scritto in Swift 6.2, progettato per gestire milioni di connessioni logiche con architettura modulare: WAL, MVCC, B+Tree e CLI amministrativa completa.

Swift 6.2 macOS 13+ Apple Silicon WAL • MVCC • B+Tree
⭐ Star su GitHub 🚀 Quick Start

✨ Caratteristiche Principali

🗄️

Storage Enterprise

Sistema di storage avanzato con heap file paginati, buffer pool LRU/Clock e compattazione online senza downtime.

  • Heap file storage con slot directory
  • Buffer pool intelligente con eviction LRU
  • Compattazione online in tempo reale
  • Ottimizzazioni Apple Silicon native
🔒

Durabilità Garantita

Write-Ahead Logging v2 con checksum CRC32, recovery ARIES-like e sistema di checkpoint efficiente.

  • WAL v2 con record tipizzati
  • Checksum CRC32 per integrità
  • Recovery ARIES-compliant
  • Gestione Dirty Page Table
🚀

Indicizzazione Avanzata

B+Tree persistenti su disco con supporto per indici pluggabili e operazioni bulk ad alte prestazioni.

  • B+Tree persistenti con checkpoint
  • Indici pluggabili (Hash, ART, LSM)
  • Validazione e manutenzione online
  • Caching intelligente per dataset grandi
⚡

Controllo Concorrenza

MVCC con livelli di isolamento configurabili, lock manager avanzato e supporto Two-Phase Commit.

  • MVCC con snapshot isolation
  • Lock manager con deadlock detection
  • Supporto 2PC per transazioni distribuite
  • Livelli di isolamento configurabili
🧠

Query Processing

Volcano iterator con planner cost-based, predicate pushdown e supporto SQL completo con sintassi moderna.

  • Planner cost-based intelligente
  • Operatori Volcano ottimizzati
  • Viste materializzate
  • Query plan cache
🛠️

Operazioni Complete

CLI amministrativa completa, import/export CSV, metriche Prometheus e policy engine per manutenzione automatica.

  • CLI amministrativa `coldb`
  • Import/export CSV con validazione
  • Metriche Prometheus integrate
  • Policy engine per ottimizzazione

🚀 Inizia in 30 Secondi

Colibrì DB è progettato per essere semplice da installare e utilizzare. Con pochi comandi puoi avere un database completo in esecuzione sul tuo Mac.

📖 Guida Completa 📚 API Reference 
# Clona e compila
git clone https://github.com/gpicchiarelli/Colibri-DB.git
cd Colibri-DB
swift build

# Avvia la CLI
.build/debug/coldb

# Crea una tabella e inserisci dati
\create table users
\insert users id=1,name=Alice,age=25
\select * FROM users WHERE name = 'Alice'
